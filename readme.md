Starter framework for building a REST API in Go

## To build
```$ make build```

## To run
```$ make run```

## How to use API
The API can be accessed from: http://127.0.0.1:8080

Request:
```
curl --location --request GET 'http://127.0.0.1:8080/api/health'
```
